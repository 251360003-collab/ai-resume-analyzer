# AI Resume Analyzer

This project analyzes a resume and detects technical skills automatically.

## Features

- Detects skills from resume text
- Generates resume score
- Suggests missing skills
- Simple AI-based analysis

## Technologies Used

- Python

## How to Run

```bash
python resume_analyzer.py
```

- Python
- SQL

- CSS

Recommended Skills:
- Java
